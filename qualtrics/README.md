# Qualtrics HTML snippets

Paste either snippet into a Qualtrics Descriptive Text question using HTML/source view.

- `qualtrics-no-reviews.html`: Phase 1 browsing page with the relisted-hotel cover story and no review content.
- `qualtrics-with-reviews.html`: Phase 2 Full Reviews Control page with ratings and short real review excerpts.

These snippets are display-only and intentionally do not include a nested `<form>` tag. Qualtrics already wraps survey pages in its own form; use native Qualtrics questions after the snippet to capture hotel choice, prior beliefs, confidence sliders, WTP, and manipulation checks.
