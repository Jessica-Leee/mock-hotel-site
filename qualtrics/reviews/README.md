# Generated hotel reviews

This folder contains synthetic review data for the six Chicago hotels in the Qualtrics snippets.

- `hotel_reviews_100_each.json`: all 600 reviews grouped by hotel.
- `hotel_reviews_100_each.csv`: all 600 reviews in spreadsheet-friendly format.
- `ground_truth_attributes.json`: provisional attribute profile used to generate review patterns.
- `<hotel_id>_reviews.json`: one 100-review JSON file per hotel.

Each review is 2-5 sentences and avoids the phrase "ground truth." Review texts are synthetic and designed for experiment use.
